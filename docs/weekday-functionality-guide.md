# 曜日表示設定機能 ユーザーガイド

## 概要

ウィークリータスクボードの曜日表示設定機能を使用すると、必要な曜日のみを表示してタスク管理を効率化できます。土日を非表示にして平日のみでの作業管理や、特定の曜日を一時的に非表示にしてスケジュール調整を行うことができます。

## 主な機能

### 1. 曜日の表示/非表示切り替え

**場所**: ヘッダーの「表示する曜日」セクション

**使い方**:
1. ヘッダーにある曜日チェックボックス（月・火・水・木・金・土・日）を確認
2. 非表示にしたい曜日のチェックを外す
3. 表示したい曜日のチェックを入れる
4. 変更は即座に反映され、設定は自動保存されます

**効果**:
- チェックを外した曜日の列が非表示になります
- その曜日に割り当てられていたタスクは自動的に「未割り当て/持ち越し」エリアに移動します
- グリッドレイアウトが動的に調整され、表示中の曜日のみが表示されます

### 2. タスクの一括移動機能

**場所**: 各曜日列のヘッダー（右クリックメニュー）

**使い方**:
1. 移動したい日の曜日列ヘッダー（例：「月曜日 (1/15)」）を右クリック
2. コンテキストメニューから「📤 全タスクを未割り当てに移動」を選択
3. その日に割り当てられた全タスクが未割り当てエリアに移動します
4. 移動したタスク数が通知で表示されます

**効果**:
- 特定の日のスケジュールを一括でクリアできます
- 急な予定変更時の調整が簡単になります
- 移動したタスクは後で他の日に再配置できます

### 3. 設定の永続化

**自動機能**:
- 曜日表示設定は自動的にLocalStorageに保存されます
- ページを再読み込みしても設定が保持されます
- エクスポート/インポート機能でも設定が含まれます

## 使用例

### 例1: 平日のみの表示設定

**シナリオ**: 土日は作業しないため、平日のみでタスク管理したい

**手順**:
1. 「土」と「日」のチェックボックスを外す
2. 月〜金曜日のみが表示される
3. 土日に割り当てられていたタスクは未割り当てエリアに移動

**結果**: 
- スッキリとした平日のみのビューでタスク管理
- 週末のタスクは未割り当てエリアで管理

### 例2: 急な会議日の調整

**シナリオ**: 水曜日に急な会議が入り、その日のタスクを他の日に移したい

**手順**:
1. 水曜日列のヘッダーを右クリック
2. 「全タスクを未割り当てに移動」を選択
3. 水曜日のタスクが全て未割り当てエリアに移動
4. 未割り当てエリアから他の日にドラッグ&ドロップで再配置

**結果**:
- 水曜日のスケジュールが一括でクリア
- タスクを他の日に柔軟に再配置可能

### 例3: 週の途中からの管理

**シナリオ**: 水曜日から週の管理を始めたい（月火は非表示）

**手順**:
1. 「月」と「火」のチェックボックスを外す
2. 水〜日曜日のみが表示される
3. 月火のタスクは未割り当てエリアで確認可能

**結果**:
- 週の後半に集中したビュー
- 過去のタスクは未割り当てエリアで管理

## 視覚的な効果

### アニメーション
- 曜日列の表示/非表示切り替え時にスムーズなアニメーション
- 列が消える時は縮小しながらフェードアウト
- 列が現れる時は拡大しながらフェードイン

### レイアウト調整
- 表示曜日数に応じてグリッドが自動調整
- 7列 → 5列 → 3列など、動的にレイアウト変更
- モバイル表示では1列に統一

### 通知システム
- タスク移動時に移動件数を通知
- 設定変更時の視覚的フィードバック
- エラー発生時の適切な通知

## 技術仕様

### パフォーマンス
- 曜日切り替え: 500ms以内
- 一括移動処理: 1秒以内  
- 設定保存: 100ms以内
- UI応答性を維持

### 互換性
- 既存のタスクデータとの完全互換性
- カテゴリフィルター機能との連携
- ドラッグ&ドロップ機能との統合
- エクスポート/インポート機能対応

### データ整合性
- 非表示曜日のタスクは自動的に未割り当てに移動
- 設定変更時のデータ検証
- エラー時のフォールバック機能

## トラブルシューティング

### Q: 曜日を非表示にしたらタスクが消えた
**A**: タスクは消えていません。「未割り当て/持ち越し」エリアに自動移動しています。曜日を再表示すると、未割り当てエリアから再配置できます。

### Q: 設定が保存されない
**A**: ブラウザのLocalStorageが無効になっている可能性があります。ブラウザの設定を確認するか、プライベートモードを解除してください。

### Q: アニメーションが重い
**A**: 大量のタスクがある場合、アニメーションが重くなることがあります。不要なタスクをアーカイブするか、カテゴリフィルターで表示タスクを絞り込んでください。

### Q: 右クリックメニューが表示されない
**A**: 日付列のヘッダー部分（曜日名の部分）を右クリックしてください。タスクエリアではメニューは表示されません。
